1. 配列と連想配列

◆ 配列とは

順序をつけて値を並べたデータを配列と言い、変数に格納が可能です。
値を要素、要素の位置をインデックス(添字)と呼びます。

上記の各 fruits 要素は[]角括弧に数値を入れて指定できます。ここでは、
インデックスの1 番目が[0]で表されている点に注意しましょう。


● 配列の記述方法

配列は主に以下の方法で記述します。

fruits = ["Apple", "Orange", "Banana", "Peach"];

この時、要素が文字列型であれば' ' や " " で囲みます。
また要素毎に , で区切る点に注意しましょう。
格納された配列は、変数[] にインデックス番号を記述する事で指定できます。 

fruits = ["Apple", "Orange", "Banana", "Peach"];
console.log(fruits[0]);
// Apple が出力




◆ プロパティとメソッドの使い方

配列が格納された変数に対し、プロパティ または メソッド名を記述し、
値の取得や要素の変更が可能です。以下はその一例です。

  • 変数.length：配列の長さ(要素の数)を取得する
  • 変数.unshift()：先頭に要素を追加する
  • 変数.push()：末尾に要素を追加する
  • 変数.sort()：昇順に並び替える

◆ 連想配列

配列のインデックス部分に対し、キーを設定した配列の事です。
値のプロパティを含めてデータを格納する事ができます。

● 連想配列の記述方法

変数 = {
  キー: 値,
};

配列は[]角括弧で囲みましたが、連想配列は{}波括弧を使用します。
キーと値の間には:コロンを記述し、値を出力する場合は変数[]の中にキーを指定します。
また、連想配列は記述が長くなりやすいので、要素毎に改行する事が多いです。

murao = {
  age: 34,
  campany_number: 352,
  birth_place: "Osaka",
  address: "Chiba",
};
console.log(murao["campany_number"]);
//352が表示される

2. オブジェクト

◆ プロパティ、メソッド、オブジェクト

この章では、オブジェクト、プロパティ、メソッドについて、
あらためて理解を深めて行きましょう。


● オブジェクト

元々は「物・事」「対象」の意味を持つ単語で、特に IT 用語としては、
プロパティの集合体を指し、プログラム上で「モノ」を表現しています。



● プロパティ

プロパティとは、名前(キー)と値がセットになったデータを指し、
オブジェクトの状態や属性を表します。値には
文字、数値、論理値、配列、関数を設定できます。

りんご をオブジェクト(モノ)とした時、
そのプロパティ(状態・属性)は次のように表せます。

種類：果物
色：赤
味:甘い
重さ:100g
組成：['皮', '果肉', '種', '果汁']




データ上の りんごは、種類や色などの情報で表現できるため、
「オブジェクト = プロパティの集まり」であると言えます。

前章で学んだ連想配列は、オブジェクトそのものでもあります。
例えば 上記の りんご オブジェクトは、以下の様に記述できます。

apple = {
  category: "fruit",
  color: "red",
  weight: "100g",
  taste: "stweet",
  structure: ["peel", "pulp", "seed", "juice"],
};

console.log(apple["structure"]);

● メソッド

プロパティを構成する「キーと値」の中で、値が関数になっているものです。
そのため、メソッドはプロパティの一部であり、更に関数とメソッドは
意味や働きとしては非常に近いものになります。


もし、りんごに対して「皮をむく」というメソッドが実行されれば、
「配列 structure から、peel 要素が削除される」 と考える事が出来ます。



3. 関数の定義(ユーザ定義関数)

◆ 関数

関数とは特定の処理を定義し、呼び出す事で実行機能を有したものです。
Excel の関数の様に、与えられた値(引数)に対して結果を返す処理が主となります。


◆ ユーザ定義関数

目的の処理を実現するため、名前や処理の内容を
自分で定義する関数を、ユーザ定義関数と言います。
また、標準で使用できる関数の事を組み込み関数といいます。


● ユーザ定義関数の書き方

ここでは function 関数の書き方を学んで行きましょう。
下記のコードが基本の形となります。

function 名前() {
  実行処理;
}
名前();

まず function を宣言し、関数の名前を指定します。
続いて{}波括弧の中に実行したい処理を記述します。

{}波括弧の外で関数名を記述すると、定義した関数を呼び出します。
関数呼び出しの()丸括弧には; セミコロンが必要なので、注意しましょう。


次の演習で、構造の感覚を掴みましょう。

演習 1
下記のコードを記述し、実行して結果を表示させましょう。
また、定義した関数名の呼出し greeting(); をコメントアウトし、結果を確認しましょう。
function greeting() {
  console.log("Hello");
}
greeting();

● 引数と戻り値

演習 1 は関数を呼び出すと、Hello の文字が出力されました。
このような処理の場合、値を渡す必要がないため、
呼び出し時の()丸括弧は空欄となります。
処理によっては値を代入し、処理を行った結果を返す必要があります。

この時、渡した値を引数、返って来た結果を戻り値といいます。

function 名前(変数) {
  変数を用いた処理;
}
名前(引数);

function 名前(変数1, 変数2) {
  変数を用いた処理;
}
名前(引数1, 変数2);

4. アロー関数

◆ アロー関数とは

function を宣言せずに関数を定義する記述方法で、
関数名の後ろの => の形から、アロー関数と呼ばれています。

基本構造は、以下の形となります。

名前 = (引数) => {
  実行処理;
};

さらに、実行する処理が 1 行の場合は{}波括弧が不要となり、
引数が 1 つだけなら()丸括弧も不要です。

名前 = (引数) => 実行処理;

引数がない場合は空白になってしまうため、丸括弧の記述が必要となります。

名前 = () => 実行処理;

記述方法に複数のパターンがありますので、
実際にコードを記述して、感覚を掴んでいきましょう。


演習 1
下記のコードを記述し、実行結果を確認しましょう。 

//引数なし
let greeting = () => console.log("Hello");
greeting();

//引数が 1つ
let square = (num) => console.log(num * num);
square(10);

//引数と処理が複数
let average = (num1, num2, num3) => {
  sum = num1 + num2 + num3;
  result = sum / 3;
  return result;
};
console.log(average(1, 2, 3));
